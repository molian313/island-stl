# Product

## Register

product

## Users

3D 打印玩家。在打印过程中需要随时查看打印进度、系统资源（内存、网速），同时希望快速启动常用文件夹/程序。使用场景：电脑前工作时瞥一眼屏幕顶部即可掌握所有状态，无需切换窗口。

## Product Purpose

macOS Dynamic Island 风格的 Windows 桌面浮窗。将系统监控（时间、内存、网络）和 3D 打印机状态（进度、温度）集成在一个紧凑的胶囊 UI 中，支持快捷方式启动。核心价值：零干扰的状态感知。

## Brand Personality

现代、流畅、精致。动效要有弹性但不夸张，视觉上像系统组件一样自然融入桌面，细节上追求像素级精确。

## Anti-references

- 避免卡片堆砌：不要用大量卡片式布局，胶囊本身就是信息载体
- 避免臃肿的 widget 风格：不要模仿 Windows 11 widgets 的大面板设计
- 避免过度装饰：不要添加无意义的渐变、光效、纹理

## Design Principles

1. **瞥一眼就够** — 信息密度高但不拥挤，核心数据一目了然
2. **动效即反馈** — 每个动画都有意义，弹性曲线传达状态变化
3. **隐形存在** — 不需要时不干扰，需要时立即响应
4. **像素精确** — 间距、圆角、对齐都经过推敲，没有"差不多"

## Accessibility & Inclusion

- 遵循 `prefers-reduced-motion` 减少动效
- 高对比度配色（深色背景 + 白色/亮色文字）
- 字号层级清晰（时间17px、状态11px、标签8px）
